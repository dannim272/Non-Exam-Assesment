## Installation
### Program - download the program repo

```bash
git clone https://github.com/dannim272/Non-Exam-Assesment.git
```

### Libraries
pip:

```bash
pip install tkinter, sqlite3, smtplib, requests, sys, time
```


pip3:

```bash
pip3 install tkinter, sqlite3, smtplib, requests, sys, time
```

### Execution
python:

```bash
python terminal.py
```


python3:

```bash
python3 terminal
```
